# kibana-fields

`kibana-fields` is an auto-generated Go library providing strongly-typed constants, label converters, type metadata, and validation helpers for the [Elastic Common Schema (ECS)](https://www.elastic.co/guide/en/ecs/current/ecs-field-reference.html) version `<VERSION>`.

Auto-generated by [`kibana-fields-generator`](https://github.com/iostrovok/kibana-fields-generator) from [`ecs_nested.yml`](https://github.com/elastic/ecs/blob/<VERSION>/generated/ecs/ecs_nested.yml).

## Installation

```bash
go get github.com/iostrovok/kibana-fields
```

## How to Use

### 1. Using Field Constants

Each ECS field set namespace (`agent`, `host`, `http`, `process`, `user`, etc.) is available as a subpackage under `github.com/iostrovok/kibana-fields/x/<fieldset>`.

```go
package main

import (
	"fmt"

	"github.com/iostrovok/kibana-fields"
	"github.com/iostrovok/kibana-fields/x/agent"
	"github.com/iostrovok/kibana-fields/x/host"
	"github.com/iostrovok/kibana-fields/x/http"
)

func main() {
	// Access strongly-typed ECS field constants
	fmt.Println(agent.ID)           // "agent.id"
	fmt.Println(agent.Name)         // "agent.name"
	fmt.Println(host.Hostname)      // "host.hostname"
	fmt.Println(http.RequestMethod) // "http.request.method"

	// Convert field to string
	var f fields.Field = agent.ID
	fmt.Println(f.String()) // "agent.id"

	// Convert field to Kibana label format (replaces '.' with '_')
	fmt.Println(f.Label()) // "agent_id"
}

```

### 2. Listing All Fields in a Field Set

Every subpackage exports a `Fields` slice containing all available `fields.Field` constants in that namespace:

```go
package main

import (
	"fmt"

	"github.com/iostrovok/kibana-fields/x/cloud"
)

func main() {
	for _, f := range cloud.Fields {
		fmt.Printf("Field: %s, Label: %s\n", f.String(), f.Label())
	}
}
```

### 3. Validating ECS Fields

Use the `check` subpackage to quickly verify whether a given string is a valid ECS field:

```go
package main

import (
	"fmt"

	"github.com/iostrovok/kibana-fields/check"
)

func main() {
	// check.Check is a map[string]bool containing all known ECS fields
	if check.Check["agent.id"] {
		fmt.Println("agent.id is a valid ECS field")
	}
}
```

### 4. Field Type Metadata

Each field set subpackage provides a `Types` struct containing Kibana data type information (`fields.KeyWord`, `fields.IP`, `fields.Date`, etc.):

```go
package main

import (
	"fmt"

	"github.com/iostrovok/kibana-fields/x/agent"
)

func main() {
	// Check underlying field type
	fmt.Printf("Type of agent.id: %T\n", agent.Types.ID) // fields.KeyWord
}
```
